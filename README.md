# Student Manager CLI

A simple command-line application built with Python to manage student records.

##  Features

- Add a student
- View all students
- Delete a student
- Persistent storage using a text file

##  Tech Stack

- Python 3
- Git for version control

##  Project Structure

student_manager/
│
├── main.py
├── students.txt
├── .gitignore
└── README.md


##  How to Run

1. Clone the repository:

git clone https://github.com/rwothani/student_manager.git


2. Navigate into the project directory:

cd student_manager


3. Run the application:

python main.py
